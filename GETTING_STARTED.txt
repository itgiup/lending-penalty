================================================================================

✅ Tính lãi kép: Gốc × (1 + rate)^n
✅ Phạt động: Tính lại mỗi 12 tháng
✅ Tiền phạt → Khoản vay mới (tính tiếp phạt)
✅ Thanh toán từng phần
✅ Hiệu ứng animation mượt mà
✅ Responsive design (Mobile/Tablet/Desktop)
✅ Backend API sẵn sàng (Hono + Cloudflare Workers)

================================================================================
🛠️ CÔNG NGHỆ
================================================================================

Frontend:
- React 18 + Vite
- Ant Design 6
- Framer Motion (animation)
- dayjs (date handling)

Backend:
- Hono (web framework)
- Cloudflare Workers
- Cloudflare D1 (database)
